# Knowledge Forge

這個 vault 是本地知識層，不是原始資料層。

## Purpose
把 Notion 裡的原始資料、追蹤頁、閱讀紀錄、任務與 dashboard，
逐步轉成可重用的本地知識：

- evergreen notes
- playbooks
- skills
- context packs

## Read Order
當我要快速找可用知識時，優先看：

1. `06_skills/`
2. `04_playbooks/`
3. `03_notes/`
4. `01_hubs/`
5. `02_sources/`
6. `90_cache/`

## Key Folders
- `00_inbox/`：尚未整理的內容
- `01_hubs/`：主題入口
- `02_sources/`：來源索引
- `03_notes/`：提煉後知識
- `04_playbooks/`：可執行手冊
- `05_projects/`：專案資料
- `06_skills/`：給 AI / agent 用的主題包
- `07_context-packs/`：打包上下文
- `90_cache/`：來源快照，不手改
- `99_templates/`：模板

## Rules
- Notion 是原始資料 truth
- `90_cache/` 是生成檔，不手改
- `02_sources/` 是索引，不等於知識本體
- 真正要長期維護的內容放在 `03_notes/` 和 `04_playbooks/`
