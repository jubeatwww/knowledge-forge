# Knowledge Forge

這個 vault 是我的本地知識庫。

## Purpose
把 Notion 中的原始資料、追蹤頁、閱讀紀錄，轉成可重用的本地知識：
- evergreen notes
- playbooks
- skill bundles
- context packs

## Source of Truth
- Notion: 原始資料、任務、追蹤頁、dashboard
- Obsidian: 本地索引、快照、提煉後知識
- Git: 版本控管與歷史

## Folder Guide
- `00_inbox/`: 尚未整理的內容
- `01_hubs/`: 主題入口頁
- `02_sources/`: Notion 來源索引
- `03_notes/`: 提煉後的 evergreen notes
- `04_playbooks/`: 可執行手冊
- `05_projects/`: 專案型資料
- `06_skills/`: 給 AI / agent 使用的主題化知識
- `07_context-packs/`: 打包好的上下文
- `90_cache/`: 自動生成的快照，不手改
- `99_templates/`: 模板

## Start Here
1. 先看 `INDEX.md`
2. 依主題進入 `01_hubs/`
3. 要找可直接使用的知識，優先看 `03_notes/` 與 `04_playbooks/`
