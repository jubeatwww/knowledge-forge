# Personal Task Center

## Summary
個人任務中心與行動啟動頁。  
核心規則是：想開 side project 前，先完成一個最小可行行動（MVA），例如看一根 K、讀一句英文、寫一行字、讀一小段 system design。

## Why this source matters
這頁不是長期知識本體，而是操作規則來源。  
最有價值的部分是：
- MVA 設計原則
- side project pre-action gate
- 低啟動成本的日常結構

- 想回看 MVA 原始設計
- 想整理 side project 啟動規則
- 想從生活流程抽出 playbook
- 想找 routines 類 evergreen notes
