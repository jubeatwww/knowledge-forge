# Skills

這裡放給 agent 直接使用的主題包。
不是每個 domain 都需要 skill pack，只有在某主題已經有明確的 `intent / schema / protocol / rules` 時才建立。

## Available Packs
- [[habit-game/INDEX]] — 目前唯一成熟的 agent-facing skill pack

## Planned Packs
- `system-design`
  條件：先從 reading tracker / project docs 抽出穩定的 review checklist、trade-off 規則與回答框架
- `frontend`
  條件：先把常見 pitfalls、debug checklist、interview answer structure 沉澱出來
- `life`
  條件：先把 MVA、weekly reset、low-friction routine 真的抽成 notes / playbooks，再決定是否需要 agent protocol

## Rule
- 有來源，不代表要有 skill pack
- 有 note / playbook，也不一定要升級成 skill
- 只有當 agent 需要固定操作邏輯時，才建立 skill pack
