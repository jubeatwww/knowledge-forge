# Daily Life Admin Reset

## When To Use
- 生活瑣事開始堆積
- 覺得腦袋一直被未完成小事佔據
- 明明工作不是最難，卻一直被生活碎事打斷

## Goal
清掉高摩擦但低價值的生活管理噪音。

## Capture Buckets
把生活 admin 先分成四類：

- 錢
- 家務
- 健康
- 行政 / 訊息 / 預約

## Reset Sequence
1. 先集中，不要一邊想到一邊處理。
2. 分出今天一定要做、這週要做、可延後。
3. 先做會持續製造精神噪音的小事。
4. 把重複事項改成固定節點處理。
5. 能自動化或模板化的就不要再靠記憶。

## Good Output
- 一份乾淨的這週 admin list
- 一條新規則或一個新模板
- 一個以後不用再想第二次的決定
