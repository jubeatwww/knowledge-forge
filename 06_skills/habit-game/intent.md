# Habit Game Intent

## 核心思路
這不是 task list，也不是 habit tracker。是一套 **身份驅動的行為系統**：
**先想成為什麼人 → 才決定做什麼事 → 最後才配工具**。

## 想成為的身份（當前版本）
| 身份 | 一句話定義 |
|---|---|
| 🧑‍💻 頂尖工程師 | 系統設計導向，在新工作上能快速 ramp-up 並交付 |
| 🗣️ 清晰的溝通者 | 中英文口語都流暢有邏輯 |
| 📈 超越指數的投資者 | 靠研究深度而非盤感勝出 |
| 💪 身心穩定的人 | 運動、營養、睡眠、心性都穩定 |
| ✍️ 持續輸出的寫作者 | 把學到的東西轉化為內容資產 |
| 🏠 有餘裕的生活者 | 家事不堆積、關係有經營、貓照顧好、財務清楚 |
| 🎨 有程式以外輸出的人 | 吉他、畫畫/AI 生圖、做遊戲，拓展輸出維度 |

## 核心觀念
**跨域思維**：一個行為可以服務多個目標。冥想同時幫心性、交易、專注力。運動同時撐住健康和工作體力。不要把行為切成孤立任務，而是理解它們的多重回報。

## 設計原則
- **身份驅動**：先問「想成為什麼人」再問「要做什麼」
- **不排滿**：推薦 3 個最划算的下一步，不是塞滿整天
- **鼓勵 > 懲罰**：低能量日完成簡單事也算 XP
- **沒事不吵**：只在偏離時推送
- **複雜度在程式碼裡**：Notion 只管存和看，邏輯在 agent 裡

## 與舊版的差異
舊版以「應徵 Google」為里程碑，LeetCode 是 P0。
當前版本：新工作（system design 導向）已到手，LeetCode 退到 P3 或移除。
主線改為：onboarding ramp-up + 健康追蹤 + 非程式活動拓展。
